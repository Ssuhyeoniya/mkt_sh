# Marketing CRM · 재구축

마케팅 CRM 보드를 새롭게 구축하는 프로젝트입니다. n개의 구글 스프레드시트를 중앙 GAS 프록시로 통합하여 단일 엔드포인트로 데이터를 제공하는 구조를 목표로 합니다.

## 메뉴 구조

- **메인** · 종합 데이터 대시보드
- **인바운드** · IB/OB 분석, IB/OB 종합, 세일즈맵
- **고객사 관리** · 성약, 이탈
- **제휴점** · 제휴점 리스트
- **광고** · 인앱, CORP
- **SNS** · 미리보기, 블로그, 인스타그램, 링크드인

## 와이어프레임 시안

브라우저에서 `wireframes/index.html`을 열어 3가지 시안을 비교할 수 있습니다.

- `wireframes/option-a/` · **Dark Minimal** — 좌측 사이드바, 다크 콘솔
- `wireframes/option-b/` · **Light BI** — 상단 탭, 라이트 BI
- `wireframes/option-c/` · **Bento Grid** — 카드형 그리드, 모던
