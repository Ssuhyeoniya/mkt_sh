3개 시안 공통 기능
  • 상단 GAS 프록시 URL 입력창 → 연결 시 실시간 데이터 반영, localStorage 저장
  • 4개 KPI: 총 게시물 / 누적 방문 / 주간 방문 / 유입(또는 전환율)
  • 제목 검색 + 카테고리 필터 + 정렬(최신/누적/주간/유입)
  • 썸네일은 시트의 thumbnail 컬럼 URL을 그대로 사용
  • 외부 링크는 새 창에서 열리는 ↗ 아이콘으로 통일
GAS 프록시 연결 방법
  1. gas/proxy.gs + gas/modules/blog.gs 를 GAS 프로젝트에 붙여넣기
  2. 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자
  3. 발급된 웹 앱 URL 복사 → 시안 페이지 상단 입력창에 붙여넣고 [연결]
  4. 자세한 가이드: gas/README.md
선택하신 시안을 베이스로 본 페이지를 확정하고, 다른 카테고리(인스타·링크드인·광고·제휴 등)도 같은 패턴으로 확장합니다.
