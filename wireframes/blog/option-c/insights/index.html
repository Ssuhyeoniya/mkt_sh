블로그 인사이트 원본

블로그 데이터에서 도출되는 인사이트 보드 디자인 시안 3종. 선택해주시면 해당 시안으로 인사이트 페이지를 확정합니다.

시안 A · Auto Insight Cards

자동 도출된 인사이트를 카드 그리드로 나열. 각 카드는 핵심 발견 + 미니 차트 + 신뢰도 스코어. 빠르게 액션 포인트 파악.

전체보기 새 창 ↗

시안 B · Comparative Analysis

카테고리/서비스/요일별 성과를 한 화면에 비교. 막대·라인·매트릭스 차트 조합. 다각도 비교 분석에 최적.

전체보기 새 창 ↗

시안 C · Score & Funnel

종합 성과 점수 + 4단계 퍼널(방문→주간→월간→인바운드) + 카테고리별 점수 카드. 성과 측정 중심.

전체보기 새 창 ↗
