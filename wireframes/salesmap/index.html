3개 시안 공통 기능
  • 상단 GAS 프록시 URL 입력창 → 연결 시 실시간 데이터 반영, localStorage 저장
  • 5개 KPI: 총 리드 / 금일 신규 / 최근 7일 / 서비스 종류(or UTM 추적) / 마케팅 동의(율)
  • 기업명·담당자·이메일 통합 검색 + 서비스·유입경로·기간 필터
  • 표시 컬럼 17개: NO, 제출 날짜, 서비스, 기업명, 임직원 수, 기업 주소, 담당자, 전화번호, 이메일, 유입경로, 개인정보/마케팅 동의, utm_source, utm_medium, utm_campaign, utm_content, utm_term
  • 헤더 우측 그립 핸들 드래그로 컬럼 너비 자유 조절 · 너비 설정은 브라우저 저장
  • 행 클릭 → 우측 드로어로 문의내용·WebFormID 포함 18개 필드 전체 확인
GAS 프록시 연결 방법
  1. gas/proxy.gs + gas/modules/salesmap.gs 를 GAS 프로젝트에 붙여넣기
  2. 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자
  3. 발급된 웹 앱 URL 복사 → 시안 페이지 상단 입력창에 붙여넣고 [연결]
  4. 라우트: ?action=salesmap.list · ?action=salesmap.summary
  5. 자세한 가이드: gas/README.md
선택하신 시안을 베이스로 본 페이지를 확정하고, 동일한 패턴으로 IB/OB 분석·고객사·제휴점 등으로 확장합니다.
